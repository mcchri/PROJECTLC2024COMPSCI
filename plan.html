The game

The memory game is codded into the microbit. The inputs are the buttons. To start the game, button A must be pressed. There are four patterns to be memorized. The game starts by showing level 1 and then one pattern, if the user pushes the right button then the game shows level 2 on the LED and then shows the previous pattern plus a new one. The levels and the amount of patterns keeps incrementing until the user presses the wrong pattern for the following stored pattern sequence. Here is a flow chart diagram explaining how the game works. My analog input is the light sensor and my digital input are the buttons. My analog output is the sound coming from the microbit, if you get the sequence correct a nice sound will play, if incorrect an unhappy noise will play. My digital outputs are the LEDs on the microbit displaying the numbers and text for the game.
