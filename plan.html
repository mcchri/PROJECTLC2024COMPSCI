The game

The memory game is codded into the microbit. The inputs are the buttons. To start the game, button A must be pressed. There are four patterns to be memorized. The game starts by showing level 1 and then one pattern, if the user pushes the right button then the game shows level 2 on the LED and then shows the previous pattern plus a new one. The levels and the amount of patterns keeps incruementing until the user presses the wrong pattern for the following stored pattern sequence. Here is a flow chart diagram explaining how the game works.

A score is initialised at the start and is increased every time when the correct button is pressed. When the wrong button is pushed the score is sent out to python.
